** Useful Keybindings
| keybinding  | Command            |
|-------------|--------------------|
| ~C-c o c~   | Open calendar      |
| ~C-c o t/T~ | Open vterm         |
| ~C-c o p~   | Open treemacs      |
| ~C-c n d~   | Open deft          |
| ~C-c 1 g~   | Grammar commands   |
| ~C-c 1 s~   | Spell commands     |
| ~C-c u~     | Undo commands      |
| ~C-c w~     | Workspace commands |
| ~C-x t~     | Tab commands       |
| ~C-c j~     | Ace jump           |
| ~C-c t~     | Active minor mode  |
| ~C-c t z~   | Zen mode           |
| ~C-c t m~   | Minimap mode       |
| ~C-c m e~   | eval commands      |
